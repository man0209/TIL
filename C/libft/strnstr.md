# strnstr.c
```
char *strnstr(const char *haystack, const char *needle, size_t len);
```
- 정해진 길이안에서 문자열을 찾는 함수.
- needle이 empty string이면 haystack 반환. len == 0이어도 haystack 반환.
- 문자열 없으면 NULL리턴.
- 문자열 찾으면 문자열 첫 주소 리턴
